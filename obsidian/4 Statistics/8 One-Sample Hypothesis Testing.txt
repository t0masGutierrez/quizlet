hypothesis

- claim about value of parameter
---
hypothesis test

- evaluate hypothesis by analyzing the significance of sample data
---
calculate hypothesis test

- state hypotheses
- choose significance level  
- calculate test statistic  
- reject or fail to reject null hypothesis  
- conclusion
---
null hypothesis

- value of parameter equal claim  
- statement of equality
---
null hypothesis formula

$$
\begin{align*}
H_0: \ =, \ge, \le
\end{align*}
$$
---
alternative hypothesis

- value of parameter not equal claim  
- statement of inequality
---
alternative hypothesis formula

$$
\begin{align*}
H_1: \ \ne, >, <
\end{align*}
$$
---
type I error

- rejecting null hypothesis given its true  
- false positive
---
type I error formula

$$
\begin{align*}
\alpha = \text{rejecting } H_0 \text{ | } H_0 \text{ true}
\end{align*}
$$
---
type II error

- failing to reject null hypothesis given its false  
- false negative
---
type II error formula

$$
\begin{align*}
\beta = \text{failing to reject } H_0 \text{ | } H_0 \text{ false}
\end{align*}
$$
---
power of hypothesis test

- measure of hypothesis test effectiveness
---
power of hypothesis test formula

$$
\begin{align*}
1-\beta = P(\text{rejecting } H_0 \text{ | } H_0 \text{ false})
\end{align*}
$$
---
significance level

- probability of type I error
---
significance level formula

$$
\begin{align*}
\alpha = P(\text{Type I Error})
\end{align*}
$$
---
critical z value

- normal distribution significance level
![[4 Statistics/Images/critical z value.png]]
---
critical z value formula

$$
\begin{align*}
z^*_{two} = \text{invNorm}(1-\frac{\alpha}{2}, 0, 1) \\
z^*_{left} = \text{invNorm}(\alpha, 0, 1) \\
z^*_{right} = \text{invNorm}(1-\alpha, 0, 1) \\
\alpha = \text{significance level}
\end{align*}
$$
---
critical t value

- t distribution significance level
![[4 Statistics/Images/critical t value.png|400]]
---
critical t value formula

$$
\begin{align*}
t^*_{two} = \text{invT}(1-\frac{\alpha}{2}, 0, 1) \\
t^*_{left} = \text{invT}(\alpha, 0, 1) \\
t^*_{right} = \text{invT}(1-\alpha, 0, 1) \\
\alpha = \text{significance level} \\
\text{df} = \text{degrees of freedom}
\end{align*}
$$

critical value

---
critical value hypothesis test

- state the hypotheses
- choose significance level
- find the critical value(s)
- calculate the test statistic
- if test statistic inside critical region then reject null hypothesis
- if test statistic outside critical region then fail to reject null hypothesis
---
p value

- probability of observing test statistic under the null hypothesis
---
p value method

- state the hypotheses
- choose significance level
- find the p value
- calculate the test statistic
- if p value less or equal significance level then reject null hypothesis
- if p value greater significance level then fail to reject null hypothesis
![[4 Statistics/Images/p value method.png|300]]
---
confidence interval

- range of guesses for the value of parameter
---
confidence interval method

- state the hypotheses
- choose significance level
- find critical value(s)
- construct confidence interval for the sample statistic
- if null value outside confidence interval then reject null hypothesis  
- if null value inside confidence interval then fail to reject null hypothesis
---
conclusion

- address claim by summarizing results
---
reject versus fail to reject

- reject null hypothesis if theres sufficient evidence that the value of parameter differs from claim  
- fail to reject null hypothesis if theres not sufficient evidence that support the claim
---
normality

- random sampling
- independent trials
- binomial distribution
- fixed number of trials 
- sampling without replacement
- at least 5 successes and 5 failures
---
proportion hypothesis test formula

$$
\begin{align*}
\text{1propZtest}(p_0, x, n, p_1) \\
p_0 = \text{null hypothesis} \\
x = \text{number of successes} \\
n = \text{sample size} \\
p_1 = \text{alternative hypothesis}
\end{align*}
$$
---
mean hypothesis test formula

$$
\begin{align*}
\text{tTest}(\mu_0, \bar x, s, n, \mu_1) \\
\mu_0 = \text{null hypothesis} \\
\bar x = \text{mean} \\
s = \text{standard deviation} \\
n = \text{sample size} \\
\mu_1 = \text{alternative hypothesis}
\end{align*}
$$
---
standard deviation hypothesis test formula

$$
\begin{align*}
\chi^2\text{cdf}(\text{lower, upper, df}) \\
\text{lower} = \text{test statistic} \\
\text{upper} = 1e99
\end{align*}
$$
